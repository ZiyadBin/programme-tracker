# Programme Tracker

This is the official Programme Tracker web application for Ziyad Bin.

## Project Overview

This repository contains the full-stack application for creating, assigning, tracking, and reporting on programmes across the district and its divisions.

* **Frontend:** React (Vite) + TypeScript + Tailwind CSS
* **Backend:** Python (FastAPI) + SQLAlchemy
* **Database:** PostgreSQL
